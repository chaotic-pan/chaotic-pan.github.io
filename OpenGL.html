OpenGl 3D Models + Refactoring Exercise

Overview

This project was created during one of my uni courses. The goal was to get familiar with OpenGL and it's uses and functionalities.

For this I created a few 3D-Models, which I then have some texture, a Shader and some Animation.
Some of the models and their textures I created by typing out their coordinates by hand. For others I read in an OBJ file and converted it into the according coordinates.
All models are shaded with a simple Phong-Shader, which we also had to write ourselves.
The animations are controlled by simple matrix commands, like scaling or rotating. These we also had to completely write by ourselves, since the class also focused on matrix calculations.

I used the same program later on, as a base for a Refractoring assignment. There, I practised using different Design Patterns and applying the Object-Oriented Programming Principles.
