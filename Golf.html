The Ultimate Golf-Game

Overview

This Project was created for one of my Uni courses. The class focused mostly on the physical laws and calculations necessarily for this game. Besides this, we also were introduced to p5.js, which is a JavaScript graphics library.
While some tasks were set by our prof, I decided to expand this game a little more on my own. So far I added different levels and a dark mode.
This game is still a work in progress, I'm planning to add a few more features and of course fix the little glitches which still hide within this game.
